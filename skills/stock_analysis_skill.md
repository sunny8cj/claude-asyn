# 深度公司投资分析

## 触发条件

当用户给出一个公司名称（如"科达利"、"贵州茅台"、"英伟达"）并期望获得投资价值分析时，使用本 skill。

## 工作流程

1. **信息收集** — 通过 WebSearch 搜集该公司的最新财报、研报、新闻、资金流向、机构评级等公开信息
2. **行业判定** — 根据公司主营业务判定所属行业，选择对应的行业分析模板
3. **分析输出** — 按下方报告框架逐项分析，生成完整报告
4. **辩证审视** — 最后必须从对立视角重新审视所有结论，防止主观偏向

## 行业个性化分析路径

分析前先判定公司所属行业，根据行业特点调整分析重点：

### 消费行业（食品饮料、家电、零售、服装、旅游等）

| 维度 | 重点关注 |
|---|---|
| 品牌力 | 品牌溢价、定价权、消费者心智占有率 |
| 渠道 | 线上线下渠道布局、经销商关系、渠道利润 |
| 复购率 | 消费频次、用户生命周期价值 |
| 消费升级/降级 | 价格带迁移趋势 |
| 政策 | 消费刺激政策、消费税、食品安全法规 |
| 关键指标 | 同店增长、客单价、坪效、库存周转天数 |

### 制造业（机械、化工、汽车、电池、结构件、材料等）

| 维度 | 重点关注 |
|---|---|
| 规模效应 | 产能利用率、单位成本曲线位置 |
| 客户集中度 | 大客户依赖度、客户分散化进度 |
| 产能周期 | 行业处于扩产期还是收缩期 |
| 原材料 | 大宗商品价格敏感度、上游议价能力 |
| 技术迭代 | 工艺升级节奏、设备更新周期 |
| 出海 | 海外工厂布局、关税影响、汇率敏感度 |
| 关键指标 | ROIC、产能利用率、订单在手量、存货周转 |

### 科技行业（互联网、软件、SaaS、云计算、通信设备等）

| 维度 | 重点关注 |
|---|---|
| 网络效应 | 用户增长→平台价值→更多用户的飞轮 |
| 技术壁垒 | 算法、专利、生态锁定 |
| 商业模式 | SaaS 订阅/广告/交易抽成/一次性销售 |
| 人才密度 | 核心技术人员稳定性 |
| 政策监管 | 数据安全、反垄断、内容审核 |
| 关键指标 | ARR/MRR、NDR、CAC/LTV、MAU/DAU、留存率 |

### 半导体（设计、制造、封测、设备、材料）

| 维度 | 重点关注 |
|---|---|
| 技术代差 | 制程节点/架构/良率与竞品差距 |
| 供应链 | 代工依赖（台积电/三星）、设备依赖（ASML/AMAT） |
| 周期位置 | 半导体周期（库存周期约3-4年） |
| 国产替代 | 政策支持力度、自主可控进度 |
| 地缘政治 | 出口管制、实体清单、技术封锁 |
| 关键指标 | 制程、良率、产能利用率、库存水位、ASP |

### 新能源/电力设备（光伏、风电、储能、锂电、电网设备、新能源车）

| 维度 | 重点关注 |
|---|---|
| 技术路线 | N型/P型/钙钛矿、大圆柱/方形/软包、LFP/三元 |
| 产能过剩 | 行业供需格局、出清进度 |
| 政策驱动 | 补贴退坡、强制配储、碳关税 |
| 出海 | 关税/反倾销、海外认证、地缘风险 |
| 储能 | 大电芯趋势、长时储能、商业模式变化 |
| 关键指标 | 装机量、GW/GWh 出货、度电成本、产能利用率 |

### 医药行业（创新药、CXO、仿制药、医疗器械、中药）

| 维度 | 重点关注 |
|---|---|
| 管线 | 在研管线进度、临床阶段、获批概率 |
| 专利悬崖 | 主力产品专利到期时间 |
| 集采影响 | 仿制药集采降价幅度 |
| 审批周期 | NMPA/FDA 审批进度与不确定性 |
| 医保谈判 | 年度医保谈判降价幅度 |
| 出海 | FDA 批准、BD 出海（license-out） |
| 关键指标 | 研发管线数、里程碑进展、获批率、医保覆盖率 |

### AI 行业（大模型、AI 应用、AI 芯片、AI 基础设施）

| 维度 | 重点关注 |
|---|---|
| 算力壁垒 | GPU/TPU 获取能力、算力成本 |
| 数据优势 | 训练数据质量、用户数据飞轮 |
| 模型能力 | 基准测试排名、实际场景表现 |
| 商业化 | API 收入、付费用户数、ARPU |
| 开源 vs 闭源 | 开源生态影响力、闭源定价权 |
| 政策监管 | 数据隐私、AI 安全法规、出口管制 |
| 关键指标 | Token 消耗量、活跃开发者数、API 调用量、训练算力规模 |

#### 6.2 "买股票就是买公司"视角（巴菲特/芒格/段永平）

- **生意质量**：这是否是一门好生意？赚钱模式是否清晰、可持续？
- **管理层品质**：管理层是否诚实、能干、以股东利益为先？
- **能力圈**：我是否真正理解这个行业和公司？
- **护城河持续性**：10年后这家公司是否仍然存在且更强？
- **价格与价值**：当前价格是否显著低于内在价值？是否有足够的安全边际？
- **机会成本**：这笔资金是否有更好的去处？

#### 6.3 周期视角（霍华德·马克斯）

- **行业周期定位**：当前处于周期的哪个阶段？（爆发期/调整期/成熟期/衰退期/二次成长期）
- **第一层 vs 第二层思维**：市场共识已经反映了什么？还有什么未被定价？
- **钟摆位置**：市场对该行业的乐观/悲观情绪处于极端还是均衡？
- **周期长度判断**：这个周期的驱动因素是什么？什么时候可能反转？

#### 6.4 全天候视角（瑞·达利欧）

| 宏观情景 | 对公司的影响 | 敏感程度 |
|---|---|---|
| 经济增长超预期 | — | 高/中/低 |
| 通胀上升 | — | 高/中/低 |
| 利率下行 | — | 高/中/低 |
| 利率上行 | — | 高/中/低 |

#### 6.5 杠铃策略视角（塔勒布）

- **杠铃一端判断**：该公司属于杠铃的哪一端？
  - 安全端（确定性高、下行风险低、收益中等）
  - 高风险端（爆发性强、可能归零也可能十倍）
  - 中间地带（不上不下，最危险）
- **是否符合杠铃策略**：如果买入，它应该放在杠铃的哪一端？
- **搭配建议**：为了构建杠铃组合，还需要搭配什么方向的投资？
  - 如果本股票偏安全端 → 搭配什么高赔率资产
  - 如果本股票偏高风险端 → 搭配什么低波动资产
  - 如果本股票在中间地带 → 建议避开或拆分为两端
- **黑天鹅暴露**：什么极端事件会让这笔投资损失惨重？概率和影响如何？

#### 6.6 风险收益比分析

| 情景 | 概率估计 | 目标价/市值 | 收益率 |
|---|---|---|---|
| 乐观 | — | — | — |
| 中性 | — | — | — |

**期望收益率** = Σ(概率 × 收益率)
**风险收益比** = 上行空间 / 下行风险

#### 6.7 辩证分析与风险提示（强制章节，不可省略）

**本部分必须从与前面结论对立的角度进行审视，防止主观偏见：**

- **看空论点**：如果你是做空者，你会如何攻击这家公司？列出3-5个最强做空逻辑
- **共识陷阱**：当前市场共识可能错在哪里？有哪些"大家都知道的好事"其实已经被过度定价？
- **被忽略的风险**：有哪些不性感、不常被提及、但可能致命的风险？
- **反身性风险**：股价上涨本身是否会改变基本面？（如估值过高导致并购冲动、管理层骄傲）
- **如果错了怎么办**：如果你对该公司的核心判断是错的，后果是什么？最大损失是多少？
- **证伪条件**：什么信号出现时，你应该重新审视或放弃这个投资判断？

### 七、推荐深入学习方向

根据公司分析中暴露的关键变量和不确定性，推荐3-6个具体学习方向，例如：
- 上下游产业链研究
- 宏观政策跟踪
- 技术路线对比
- 竞争格局演变
- 投资方法论对应（如某位投资大师的分析框架特别适合这类公司）

每个方向附带"为什么值得学"和"从哪里开始"。

## 信息收集要求

执行分析前，至少进行以下搜索（使用 WebSearch）：

1. **最新财报**：`公司名 最新年报 季报 分业务 分地区 营收 毛利率`
2. **竞争格局**：`公司名 竞争对手 护城河 市场份额 行业地位`
3. **估值与评级**：`公司名 估值 PE PB 机构评级 目标价`
4. **资金面**：`公司名 换手率 资金流向 北向资金 机构持仓`
5. **海外/地缘**：`公司名 海外业务 出海 关税 地缘政治`
6. **行业趋势**：`行业名 2026 市场规模 趋势 政策`
7. **管理层访谈**：`公司名 管理层 董事长 访谈 调研纪要`

## 输出原则

- 所有数据必须标注来源或注明"估算"
- 使用表格呈现结构化数据，文字用于解读和判断
- 辩证分析章节不可省略，必须包含真实的反面观点
- 不做绝对推荐，给出分析过程和结论让用户自行判断
- 结尾必须附风险提示
