# 微信公众号 HTML 导出技能

将 Markdown 分析/研究报告转换为可直接粘贴到微信公众号编辑器的 HTML 格式。

## 核心特性

- **内联样式**：所有样式使用 `style="..."`，公众号不会丢弃
- **图片内嵌**：图片自动转为 base64 内嵌，复制时图片一起粘贴
- **颜色体系**：标题蓝色 `#1a73e8`、小节橙色 `#f39c12`、关键数据红色 `#e74c3c`、正面绿色 `#27ae60`
- **高清输出**：图表 DPI 300，字号 15px 适配手机

## 工作流程

当用户触发此技能时，按以下步骤执行：

### Step 1：确认输入源

询问或自动查找最近的 Markdown 文档。通常位于 `industry-analysis/` 目录。

### Step 2：运行转换脚本

```bash
python "C:\Users\75307\.claude\skills\wechat-html-export\wechat_html_export.py" \
  "<md文件路径>" \
  --img-dir "<图片目录路径>" \
  --title "<标题> | <副标题>" \
  --subtitle "数据截至：YYYY-MM-DD" \
  --tag "自定义标签文字"
```

### Step 3：告知用户使用方式

生成完毕后告知用户：
1. 用浏览器打开生成的 HTML 文件
2. `Ctrl+A` 全选，`Ctrl+C` 复制
3. 打开公众号编辑器，`Ctrl+V` 粘贴
4. 图片和样式会完整保留

## 脚本参数

| 参数 | 必填 | 说明 | 默认值 |
|------|------|------|--------|
| `input` | 是 | 输入 Markdown 文件路径 | - |
| `--img-dir` / `-i` | 否 | 图片目录路径 | `{input_dir}/images` |
| `--output` / `-o` | 否 | 输出 HTML 路径 | `{input_basename}.html` |
| `--title` / `-t` | 否 | 文档标题 | 文件名 |
| `--subtitle` / `-s` | 否 | 副标题 | 空 |
| `--tag` | 否 | 标签徽章文字 | 空 |
| `--disclaimer` / `-d` | 否 | 底部免责声明 | 标准投资免责声明 |

## 输出样式说明

| Markdown 元素 | HTML 样式 |
|---|---|
| `# 一级标题` | 居中蓝色 26px |
| `## 二级标题` | 左侧蓝色色条 + 蓝色文字 17px |
| `### 三级标题` | 橙色文字 15px |
| `**加粗**` | 加粗保留 |
| `*斜体*` | 斜体保留 |
| `| 表格 |` | 蓝色表头 + 斑马纹行 |
| `> 引用` | 黄色背景引用框 |
| `![图片](path)` | base64 内嵌 + 圆角 100% 宽 |
| `---` | 灰色分隔线 |

## 注意事项

- 图片文件必须存在（PNG/JPG），缺失图片会显示占位文字
- 生成文件较大（1-3 MB），因为图片内嵌了 base64
- 如果图片不在 `{input_dir}/images/` 下，用 `--img-dir` 指定正确路径
