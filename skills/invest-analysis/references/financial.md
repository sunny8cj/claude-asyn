# 财务分析详细框架

## 盈利能力指标

### 毛利率
- **计算**：(营收 - 营业成本) / 营收
- **分产品线分析**：高毛利业务占比提升 → 整体盈利能力改善
- **行业对比**：同行业毛利率差异反映定价权和成本控制
- **趋势**：毛利率持续提升 → 护城河加深；持续下降 → 竞争加剧或成本失控

### 净利率
- **计算**：净利润 / 营收
- **扣非净利率**更能反映经营能力
- **对比毛利率**：差距大 → 三费高，关注费用控制

### ROE（净资产收益率）
- **杜邦分析拆解**：ROE = 净利率 × 资产周转率 × 权益乘数
- **三种驱动模式**：
  - 净利率驱动 → 高利润型企业（茅台）
  - 周转率驱动 → 高周转型企业（沃尔玛）
  - 杠杆驱动 → 高杠杆型企业（银行）—— 需警惕风险
- **连续 5 年 ROE > 15%** 通常表明竞争优势

## 成长性指标

### 营收增速
- 同比增速（YoY）、环比增速（QoQ）
- 关注增速拐点（加速/减速）
- 有机增长 vs 并购增长

### 利润增速
- 扣非归母净利润增速
- 增速是否高于营收增速 → 经营杠杆效应

## 现金流分析

### 经营现金流
- **经营现金流/净利润** > 1 → 盈利质量高
- 持续 < 1 → 可能存在应收账款膨胀或利润调节

### 自由现金流
- **计算**：经营现金流 - 资本开支
- 正向且稳定 → 公司内生增长能力强
- 持续为负 → 依赖外部融资

### 现金流类型判断
| 经营 | 投资 | 筹资 | 类型 |
|------|------|------|------|
| + | - | - | 成熟奶牛型 |
| + | - | + | 稳健扩张型 |
| - | - | + | 初创投入型 |
| - | + | + | 拆东补西型（警惕） |

## 资产负债表分析

### 关键比率
- **资产负债率**：行业差异大，关注同业对比
- **流动比率** > 1.5、**速动比率** > 1 → 短期偿债能力 OK
- **有息负债率**：有息负债/总资产，高负债需关注利息覆盖倍数

### 应收账款
- **应收账款周转天数**趋势：延长 → 回款恶化
- **账龄结构**：1 年以上占比上升 → 坏账风险
- **应收/营收比**：>30% 需警惕

### 存货
- **存货周转天数**：延长 → 滞销风险
- **存货跌价准备**计提是否充分

## 成本结构分析

### 三费占比
- **销售费用率**：品牌型公司较高，但需看趋势
- **管理费用率**：持续上升 → 管理效率下降
- **研发费用率**：科技/医药公司核心指标，需看资本化比例

### 成本控制能力
- 三费占比与行业对比
- 期间费用率趋势（3-5 年）
- 规模效应是否体现
