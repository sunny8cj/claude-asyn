# 宏观环境分析要素

## 地缘政治影响

### 评估维度
- 公司业务覆盖国家/地区
- 主要客户所在国政治稳定性
- 供应链地理分布风险

### 高风险信号
- 业务高度依赖单一地区（如俄乌、中东）
- 关键原材料来自敏感国家
- 大客户所在国政策突变

## 汇率影响

### 分析框架
- 海外营收占比
- 主要结算货币（美元/欧元/人民币）
- 海外成本结构

### 汇率敏感度
- **出口型企业**：人民币贬值利好
- **进口依赖型**：人民币升值利好
- **套期保值**：是否有汇率对冲措施

### 跟踪指标
- 美元指数走势
- 人民币汇率中间价
- 公司汇兑损益历史

## 美联储政策影响

### 关注要点
- **加息周期**：
  - 全球流动性收紧
  - 高估值成长股承压
  - 新兴市场资金流出
  
- **降息周期**：
  - 流动性宽松
  - 风险偏好提升
  - 成长股估值修复

### 传导路径
```
美联储政策 → 全球流动性 → 风险偏好 → 股市估值
```

### 跟踪数据
- 联邦基金利率
- 美联储点阵图
- 美债收益率曲线

## 关税与贸易政策

### 影响分析
- 主要出口市场关税政策
- 原产地规则影响
- 贸易摩擦受益/受损

### 典型案例
- **光伏/新能源车**：欧美关税壁垒
- **消费电子**：关税豁免清单变化
- **半导体**：出口管制升级

### 跟踪渠道
- 商务部公告
- 海外贸易政策新闻
- 公司投资者关系回复

- **中性**：规范化、标准化

## 宏观周期判断

### 经济周期
- GDP 增速趋势
- PMI、CPI、PPI 走势
- 社融、M2 增速

### 行业周期
- 行业景气指数
- 产能利用率
- 库存周期位置
