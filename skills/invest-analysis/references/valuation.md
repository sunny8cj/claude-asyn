# 估值分析方法

## 内在价值估算

### DCF 法（现金流折现）
**适用**：现金流稳定、可预测性强的企业

**核心公式**：
```
内在价值 = Σ(未来自由现金流 / (1+WACC)^t) + 终值
```

**关键假设**：
- 未来 5-10 年自由现金流预测
- WACC（加权平均资本成本）：通常 8-12%
- 永续增长率：通常 2-4%

**敏感性分析**：做多情景测算（乐观/中性/悲观）

### PE 估值法
**适用**：盈利稳定的非周期股

**估值维度**：
- PE(TTM) 历史分位（5 年）
- 行业平均 PE、龙头 PE
- 合理 PE 区间判断

**PE Band**：
```
合理股价 = EPS × 合理PE
保守目标 = EPS × 历史PE下限
乐观目标 = EPS × 历史PE上限
```

### PB 估值法
**适用**：重资产、周期股、银行保险

**关注**：
- PB 历史分位
- ROE 与 PB 匹配关系（高 ROE 应享高 PB）

### PS 估值法
**适用**：尚未盈利的成长股

**逻辑**：
- 高增速可享高 PS
- 关注 PS 与增速匹配

### PEG 法
**适用**：高成长股

```
PEG = PE / 增速(%)
PEG < 1 → 低估
PEG = 1 → 合理
PEG > 1 → 高估
```

## 历史估值分位

**计算方法**：
- 取近 5 年 PE/PB 数据
- 计算当前值在历史区间的百分位
- 分位 < 30% → 低估值区域
- 分位 > 70% → 高估值区域

## 资金面分析

### 机构持仓
- 北向持股比例及变化
- 公募基金持仓变化
- 社保/险资持仓

### 流动性
- 流通市值、换手率
- 成交额、量价关系
- 融资融券余额变化

### 事件时间表
- 解禁计划（解禁比例、解禁日期）
- 增持/减持公告
- 回购计划

## 估值判断输出

```
估值区间：[价格下限] - [价格上限]
当前价格：[现价]
偏离度：[高估/合理/低估] X%
安全边际：[充足/一般/不足]
```
