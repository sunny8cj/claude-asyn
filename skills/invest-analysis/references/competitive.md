# 竞争分析框架

## 护城河识别

巴菲特护城河四要素：

### 1. 无形资产
- **品牌溢价**：是否可以定价更高？（茅台、爱马仕）
- **专利壁垒**：专利数量、有效期、技术护城河
- **牌照垄断**：特许经营权（免税牌照、金融牌照）
- **判断标准**：能否让竞争对手无法复制或复制成本极高

### 2. 转换成本
- **客户粘性**：更换供应商的隐性成本
- **典型案例**：企业软件（SAP）、银行账户、手机生态
- **判断标准**：客户流失率低、续约率高

### 3. 网络效应
- **用户规模价值**：用户越多产品越有价值
- **典型案例**：微信、淘宝、美团
- **判断标准**：新用户边际成本趋零、平台价值指数增长

### 4. 成本优势
- **规模效应**：规模越大单位成本越低
- **独特资源**：矿产、地理位置、稀缺供应链
- **流程优势**：精益管理、技术领先

## 波特五力分析模板

### 1. 供应商议价能力
- 供应商集中度
- 原材料替代性
- 公司采购占比
- **高风险信号**：单一供应商、不可替代原料

### 2. 客户议价能力
- 客户集中度（前五大客户占比）
- 产品差异化程度
- 客户价格敏感度
- **高风险信号**：大客户依赖 >50%

### 3. 新进入者威胁
- 行业准入门槛（资金、技术、牌照）
- 规模经济要求
- 品牌忠诚度
- **低威胁**：高资本密集 + 技术壁垒 + 牌照垄断

### 4. 替代品威胁
- 是否存在替代性产品/服务
- 替代品性价比
- 客户转换意愿
- **警惕**：颠覆性技术替代（柯达被数码取代）

### 5. 行业竞争程度
- 行业集中度（CR3、CR5）
- 价格战程度
- 产能利用率
- **理想状态**：寡头格局、差异化竞争
